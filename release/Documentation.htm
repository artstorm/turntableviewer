JS Turntable Viewer 2.0: Documentation

Here is a brief introduction to get started using the Turntable Viewer and what settings that can be used. See the included example folder for practical examples of how to use these settings.

Available Settings

The below table shows all available settings that can be used to configure the viewer. These settings can be initialized with flashvars or you can create a config XML file and just just config=path to xml file as a flashvar.

file
(Required) Path to a turntable file or a gallery XML file.
preview
(Default: n/a) Path to a preview image to display before any turntable clip is loaded.
displaycolor
(Default: n/a) The color used behind the turntable clip if it doesn't fill the entire view area
backcolor
(Default: n/a) The background color of the control bar and the color used on display icons and text.
frontcolor
(Default: n/a) The color used on the scrubbing track, the icon on the scrubber and the background of display icons.
scrubcolor
(Default: n/a) The color of the scrubber.
theme
(Default: n/a) The path to a custom theme file.
stretching
(Default: UNIFORM) How clips shall be stretched if they don't fit in the viewing area. Available options are: NONE (No stretching), UNIFORM (Fills the screen with aspect kept, adding black bars if needed), FILL (fills the view, and cuts parts outside the view), EXACTFIT (Stretches to fit the view)
thumbsize
(Default: 80) The size of the thumbnails when a gallery is used.
thumbmargin
(Default: 20) The distance between the thumbnails in a gallery.

File

If you intend to only display a single clip in the viewer you can specify the path to your turntable clip with this setting. Currently the supported clip types are either an AS3 compiled SWF containing the frames laid out on the timeline or a jpg/png/gif image sequence.

When using an image sequence, set the path to the first image in the image sequence, ie turntable/tt-sequence_0001.jpg and the viewer will find all following frames when loading the clip.

If you intend to use the viewers built in gallery feature, specify the path to your gallery XML file here instead. The XML file is setup like this:

<?xml version="1.0" encoding="utf-8"?>
<turntable>
<clip>
<thumb>thumb0.jpg</thumb>
<file>imgseq_test/imgseq_0001.jpg</file>
</clip>
<clip>
<thumb>ninja1-thumb.jpg</thumb>
<file>ninja1.swf</file>
</clip>
<clip>
<thumb>ninja2-thumb.jpg</thumb>
<file>ninja2.swf</file>
</clip>
</turntable>

ie. <clip><thumb>path to thumbnail</thumb><file>path to turntable</file></clip>

for each turntable clip in the gallery.

Colors

Colors are defined with hexadecimal numbers. ie 000000, ff0000 or aa4523.

Theme

The path to a swf file containing a custom theme for the viewer, which allows much more customization than changing the color values. See the theme folder in the archive of how to create your own theme to use.
