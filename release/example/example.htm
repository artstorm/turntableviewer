Examples

Thank you for downloading the viewer. Below you see a few examples of the embedded Turntable Viewer. Copy and paste the source on your site to get started.

Example 1 - Single Turntable

Most basic example, a preview image and a single turntable file provided.

Example 2 - Customized Colors

Same as above, but with the colors customized through an external xml file.

Example 3 - Turntable Gallery

Two turntables provided through an external xml file with thumbnail popups to select between them. The preview image is also set in the external xml.
